# Project Instructions for AI Agents

## Code Quality

- Do not introduce deprecated syntax, APIs, or framework patterns.
- When editing existing code, replace deprecated usage in the touched area with the current recommended syntax.
- For Tailwind CSS v4 important modifiers, use the modern trailing `!` syntax:
  - Use `hover:border-brand-red!`, not `hover:!border-brand-red`.
  - Use `focus:ring-1!`, not `focus:!ring-1`.
  - Use `text-white!`, not `!text-white`.
- Run the relevant build or test command after changes when practical.
