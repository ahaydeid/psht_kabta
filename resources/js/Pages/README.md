# Pages Structure

Inertia pages are grouped by area and business module.

- `Home/`: public or neutral entry pages.
- `Auth/`: authentication pages.
- `Admin/Dashboard/`: admin landing and summary pages.
- `Admin/MasterData/`: CRUD master records such as siswa, warga, pelatih, and tingkatan.
- `Admin/Operasional/`: daily workflows such as absensi and jadwal latihan.
- `Admin/Laporan/`: reporting pages.
- `Admin/Pengaturan/`: roles, permissions, and user management.
- `Mobile/`: mobile-native-like user views.
